                    		Programming Assignment #2
----------------------------------------------------------------------------------------

This assignment consist of three classes Board, Connect4AI & CS401_K142111_A2P1(Main Class).
In this assignment i implemented alpha-beta prunning in the minimax algorithm.

It first provide you with a choise of either to take the first move or let the
AI have it, and then it will be prompting you for your moves while updating the
board on every move after AIs' response.

It is simple JAVA base program so you will need to have JAVA install and have some JAVA supported 
IDE in order to run it.
